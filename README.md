# Weatherly

A full-stack weather application built with React and Node.js that retrieves current weather conditions and a five-day forecast using the OpenWeather API.

## About

This project demonstrates how a React frontend communicates with a Node.js backend to retrieve weather data from the OpenWeather API. The backend fetches current weather conditions and forecast data, transforms the API responses into a simplified format, and exposes REST API endpoints for the frontend to consume.

## Features

- Search weather by city
- Search weather using geographic coordinates
- Current weather conditions
- Five-day weather forecast
- Node.js REST API
- React frontend consuming backend endpoints
- Integration with the OpenWeather API
- Error handling for failed API requests and invalid input

## Tech Stack

- React
- Node.js
- JavaScript (ES Modules)
- Axios
- OpenWeather API
- HTML
- CSS
- Vite

## Getting Started

Install the backend dependencies:

```bash
cd backend
npm install
```

Start the backend:

```bash
npm run dev
```

In a new terminal, install the frontend dependencies:

```bash
cd frontend
npm install
```

Start the React frontend:

```bash
npm run dev
```

## Purpose

This repository was created to practise building full-stack applications by integrating a React frontend with a Node.js backend and working with a third-party REST API.
